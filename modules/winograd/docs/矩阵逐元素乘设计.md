# 矩阵逐元素乘单元 KTU (`matrix_pointwise_mult_6x6_sim.sv`)

规格如下：

1. 适用 $F(4\times4,3\times3)$ 的 Winograd 卷积算法
2. 输入输出固定为 $6\times6$
3. 传入 start 信号后开始计算
4. 输入 16 位宽，输出 32 位宽，注意值域不要超出数据范围
5. 使用 36 个 DSP 单元，并行计算，延迟为 3 个时钟周期

## 行为级仿真

`matrix_pointwise_mult_6x6_sim.sv`，测试五种输入，测试均通过

## 时序级仿真

经过测试，在 200 MHz 下可以以约 0.165ns 的余量工作
