# 卷积核预处理单元 KTU (`kernel_transform_unit.sv`)

规格如下：

1. 适用 $F(4\times4,3\times3)$ 的 Winograd 卷积算法
2. 输入卷积核固定为 $3\times3$，输出卷积核为 $6\times6$
3. 为了速度考虑，进行的是整数乘法，因此输出结果被扩大了 $6\times6=36$ 倍，后续需要缩放回来
4. 所有的运算都是 16 位宽，注意值域不要超出数据范围
5. 流水线设计，共有 3 个状态，即 3 个时钟周期完成一次卷积核预处理，transform_done 为高时表示当前卷积核预处理完成，可以读取输出数据

## 仿真文件对应

`kernel_transform_unit_sim.sv`，测试五种卷积核输入：

1. 单位中心卷积核
2. 全 1 卷积核
3. 边缘检测卷积核
4. 序列卷积核
5. Sobel 卷积核
