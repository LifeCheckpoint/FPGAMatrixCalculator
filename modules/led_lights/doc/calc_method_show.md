# Calc Method Show 模块设计文档

## 1. 概述
`calc_method_show` 模块用于在七段数码管上显示特定的字母字符（T, A, B, C, J）。该模块主要用于状态指示或特定模式下的显示需求。

## 2. 接口定义

| 信号名 | 方向 | 位宽 | 描述 |
| :--- | :--- | :--- | :--- |
| clk | input | 1 | 系统时钟 |
| rst_n | input | 1 | 异步复位，低电平有效 |
| method_sel | input | 3 | 字符选择信号 |
| seg | output | 8 | 七段数码管段选信号（低电平有效） |
| an | output | 4 | 七段数码管位选信号（低电平有效） |

## 3. 功能描述

### 3.1 字符映射
模块根据 `method_sel` 输入信号显示对应的字符：

| method_sel | 显示字符 | 段码 (seg) | 描述 |
| :--- | :--- | :--- | :--- |
| 0 | **T** | `8'b10000111` | 显示字母 T |
| 1 | **A** | `8'b10001000` | 显示字母 A |
| 2 | **B** | `8'b10000011` | 显示字母 b (小写) |
| 3 | **C** | `8'b11000110` | 显示字母 C |
| 4 | **J** | `8'b11110001` | 显示字母 J |
| 其他 | (Blank) | `8'b11111111` | 不显示 |

### 3.2 扫描显示
模块内部包含扫描逻辑，用于驱动多位共阳极七段数码管。目前设计在所有激活的位上显示相同的选定字符。
- 刷新频率：由内部计数器控制（每 50000 个时钟周期切换一次位选）。
- 位选信号 `an` 依次激活 4 个数码管。

## 4. 时序说明
- 复位后，计数器清零，从第一位开始扫描。
- `seg` 输出随 `method_sel` 的变化即时更新（组合逻辑解码）。
