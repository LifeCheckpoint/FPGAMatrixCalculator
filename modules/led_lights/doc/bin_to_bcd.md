# bin_to_bcd - 二进制转BCD码

## 功能
将16位二进制数转换为4位BCD码（支持0-9999）。

## 端口
| 端口 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| bin_in | input | [15:0] | 二进制输入 |
| bcd_out | output | [3:0][0:3] | BCD输出数组 |

## BCD输出排列
- `bcd_out[0]`: 千位
- `bcd_out[1]`: 百位
- `bcd_out[2]`: 十位
- `bcd_out[3]`: 个位

## 工作原理
1. 采用除法和取模运算转换各位数字
2. 前导零消除：将前导零转换为15（LED不显示）
3. 保留最低位，即使为0

示例：
- 输入5 → 输出[15, 15, 15, 5]
- 输入42 → 输出[15, 15, 4, 2]
- 输入305 → 输出[15, 3, 0, 5]
- 输入1234 → 输出[1, 2, 3, 4]

## 特性
- 纯组合逻辑，无延迟
- 自动前导零消除
- 自动截断超过9999的高位
